## 문제 이름과 링크

- [이친수 구하기](https://www.acmicpc.net/problem/2193)

## 풀이에 관련된 주된 이론

- DP

## 풀이 방법 (생각의 Flow 방식으로 글 작성)

- 1은 두번 연속으로 나오지 않는 조건에서 
- 이전 상태에서 현재 상태가 될수 있는 경우의 수를 모두 본다
- D[i-1]  D[i] 
     0  ->  0  
            1
     1  ->  0
     총 3가지의 경우의 수가 나온다
- D[i][0] = D[i-1][0] + D[i-1][1]
  D[i][1] = D[i-1][0]      
  위의 경우의 수를 토대로 점화식을 이용해서 구한다
